## Overview

- **View important messages to the camp from Admin**
- **See a Snapshot of Camper Data**
- **View status of equipment**
- **Responsive interface.**
- **Update, Create and Delete any items with a login**
- **Limit use by junior staffers**

## Technologies Used

- **MERN**
- **Bootstrap**
- **JWT**
- **mongoimport**
